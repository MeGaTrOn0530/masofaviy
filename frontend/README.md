# Meeting Frontend Demo

Bu frontend `backend` dagi meeting service'ni tez tekshirish uchun yozilgan React + Vite demo UI.

## Ishga tushirish

```bash
cp .env.example .env
npm install
npm run dev
```

Default URL: `http://localhost:5173`

## Demo oqim

1. `Demo Login` orqali teacher yoki student sifatida kiring.
2. Teacher bo'lsangiz meeting yarating va `Start` qiling.
3. Shu yoki boshqa brauzer oynasida student bilan kiring.
4. Student `Join` tugmasi orqali roomga kiradi.
5. Media, chat va teacher control panelni tekshirib ko'ring.
